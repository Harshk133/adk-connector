# Telegram Basic Example

This example demonstrates how to deploy a Google ADK (Agent Development Kit) agent to Telegram using ADK Connectors in just a few lines of code.

## Setup

1. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```
2. Fill in your `TELEGRAM_BOT_TOKEN` (get it from `@BotFather` on Telegram) and your `GEMINI_API_KEY` (get it from Google AI Studio).

3. Run the script:
   ```bash
   python main.py
   ```

4. Open Telegram and start chatting with your bot!
